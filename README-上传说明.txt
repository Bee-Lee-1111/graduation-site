毕业纪念网页上传说明

这个 dist 文件夹就是最终可上传的网站版本。

必须一起上传：
- index.html
- images 文件夹
- images 文件夹里的所有 JPG 图片

上传后访问方式：
- 如果上传到网站根目录，访问：https://你的域名/index.html
- 很多平台也支持直接访问：https://你的域名/

注意：
- 不要只上传 index.html，否则照片不会显示。
- 不要改 images 文件夹名称，否则照片路径会失效。
- 当前网页使用相对路径 images/DSCxxxxx.JPG，上传到同一个网站目录后就会自动变成网络地址，不需要手动逐张修改。
